"# Traductor-IA" 
## Models
Se trata de un modelo Transformer pre-entrenado llamado "MarianMTModel"

Documentation MarianMTModel : https://huggingface.co/docs/transformers/model_doc/marian


## Traducciones
Español - Inglés


Inglés - Español

Español - Alemán

Alemán - Español

## Probar el Modelo
cd traductor_ia

cd data

cd src

cd models

## Ejecutar el Modelo (carpeta Models Transformer.py)

python Transformer.py
